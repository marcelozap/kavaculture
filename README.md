# Kava Culture

Static landing and phone-ordering concept for `kavaculture.xyz`.

## Files

- `index.html`: page structure
- `styles.css`: brand styling and responsive layout
- `app.js`: interactive ordering flow and per-drink poster generation
- `assets/`: menu board references, logo, and shared background texture
- `CNAME`: GitHub Pages custom domain

## Notes

This is a front-end prototype designed for a future direct-pay integration. The checkout UI currently demonstrates the guest flow, fee comparison, and gratuity behavior without a live payment backend.
